## 對炭火誘惑的直接應用

### 🔥 馬上可用（高優先）

#### 1. Backdrop-filter 毛玻璃（新竹岩極火鍋）
```css
backdrop-filter: blur(12px) saturate(180%);
```
**應用場景**：Hero 區塊滾動時，文字卡片疊在炭火背景上，產生火鍋煙霧朦朧感

#### 2. Film Grain 膠片顆粒（昭和生活節）
```html
<canvas id="film-grain"></canvas>
<script>
const canvas = document.getElementById('film-grain');
const ctx = canvas.getContext('2d');
function renderGrain() {
  const imageData = ctx.createImageData(canvas.width, canvas.height);
  for (let i = 0; i < imageData.data.length; i += 4) {
    const val = Math.random() * 255;
    imageData.data[i] = imageData.data[i+1] = imageData.data[i+2] = val;
    imageData.data[i+3] = Math.random() * 50; // low opacity
  }
  ctx.putImageData(imageData, 0, 0);
  requestAnimationFrame(renderGrain);
}
</script>
```
**應用場景**：全頁面覆蓋淡淡的 film grain，強化「深夜炭火食堂電影感」

#### 3. 跑馬燈 Marquee（新竹岩極火鍋）
```css
@keyframes marquee {
  0% { transform: translateX(0); }
  100% { transform: translateX(-50%); }
}
.marquee-inner {
  display: flex;
  width: max-content;
  animation: marquee 20s linear infinite;
}
```
**應用場景**：菜單招牌菜名橫向無限滾動

#### 4. FAQ 折疊（原生的 details/summary）
```html
<details class="faq-item">
  <summary>魚要烤多久？</summary>
  <div class="faq-content">約15-20分鐘...</div>
</details>
```
```css
.faq-item summary {
  list-style: none;
  cursor: pointer;
}
.faq-item[open] .faq-content {
  animation: slideDown 0.3s ease-out;
}
```
**應用場景**：目前 FAQ 可升級為此設計
